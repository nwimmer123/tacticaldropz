About & Help — TacticalDropz

What it is

TacticalDropz is a browser-based planning tool for visualizing Warhammer 40K deployments, terrain layouts, and objectives before you get to the table.

Why it exists

Tournament preparation often involves rough sketches, scattered notes, and last-minute guesswork. TacticalDropz exists to make that process faster, clearer, and easier to iterate — without paper or screenshots.

Who it’s for

  • Warhammer 40K players preparing for tournaments or leagues
  • Players who prefer visual planning over written notes
  • Groups discussing deployments and terrain layouts together

What it’s not

  • A rules reference
  • An army list builder
  • A replacement for playing games on the table

Status

TacticalDropz is an active side project and continues to evolve based on real player feedback. Features and layouts may change over time.

How to Use

Getting started

Select your terrain format (GW, WTC, or UKTC) and a mission from the left sidebar. The board updates with the correct deployment zones and objectives. Use the layout dropdown in the top nav to cycle through terrain layouts.

🖊️ Draw Tool

Click to place points and outline a unit on the board.

  • Left click — add a point
  • Right click — undo last point
  • Double click — close and finish the shape
  • Drag inside a finished shape — move it

📏 Measure Tool

Click two points to measure distance in inches.

  • Left click — set start, then end point
  • Right click on a line — delete it

👁️ Line of Sight Tool

Draw a line of sight check between two points.

  • Left click — set origin, then target
  • Right click on a line — delete it

🪧 Label Tool

Place a text label anywhere on the board.

  • Left click — place label (prompts for text)
  • Right click on a label — delete it

🪖 Place Models Tool

Place model bases on the board with live cohesion checking.

  • Select base size and model count in the toolbar first
  • Left click empty space — drop a group
  • Left click a base — select it (enters edit mode)
  • Ctrl + click — add/remove a base from selection
  • Drag a selected base — move the whole selection
  • Right click a base in edit mode — remove it
  • Left click empty space in edit mode — exit edit mode
  • Cohesion lines show green when within 2" or red when out of coherency

💾 Save / Load / Clear

  • Save Plan — downloads your board as a JSON file
  • Load Plan — restores a previously saved board
  • Clear — resets the board
