What is TacticalDropz?

TacticalDropz is a browser-based planning tool for visualizing Warhammer 40K deployments before you get to the table. Import your army list, drag units onto the board, check cohesion, measure distances, and plan your deployment against real mission layouts — all without paper or screenshots.

Typical Workflow

1
Select your mission — pick from the left sidebar, set terrain format (GW or WTC) and layout in the top nav.
2
Import your army list — click 📋 Import List in the staging area. Paste from the GW App or New Recruit. Review counts and base sizes in the review modal.
3
Drag units to the board — drag unit pills from the staging area and drop them onto the board. Units are placed as correctly-sized model bases.
4
Adjust positions — click a base to enter edit mode, drag to reposition, rotate ovals and rectangles with the ↻ handle.
5
Save or export — Pro users save to cloud. Free users can download a JSON snapshot with Save Plan.

📦 Staging Area

The staging area sits between the toolbar and the board. It shows all units from your imported list as draggable pills.

  • Each pill shows unit name, model count, base size, and a color dot by category
  • Red = Characters    Teal = Battleline    Blue = Vehicles    Purple = Monsters    Orange = Transports
  • Deployed units grey out with a dashed border — drag again to reposition on the board
  • Deleting a unit from the board returns it to staging automatically
  • Clear Board — removes all drawings, returns list units to staging (list stays intact)
  • Clear List — removes the imported list entirely and clears all list-placed units
💡 You can mix imported units with hand-placed units. Right panel shows a green ✓ for list units and yellow M for manually placed ones.

📋 List Import

TacticalDropz supports two list formats:

GW App (Official App export)

Copy your list from the GW App and paste it directly. The parser extracts unit names, counts models by counting bullet points (not weapons), and auto-infers base sizes. Works even when the app exports everything as a single long line.

New Recruit — Short format (recommended)

Uses the pattern 10x Unit Name (pts). Model counts are read directly from the Nx prefix — most reliable format.

Review Modal

After pasting, a review modal shows every parsed unit with editable count, shape picker, and filtered size dropdown. Correct anything before units hit staging.

  • Select shape first — ○ circle, ⬭ oval, ▭ rectangle — then pick the dimension from the filtered list
  • ✓ = base size confident    ? = inferred, please verify

🎯 Place Models Tool

The default tool. Place and arrange model bases directly on the board.

Toolbar controls

  • Shape picker — select ○ circle, ⬭ oval, or ▭ rectangle first. The size dropdown filters to matching options only.
  • Size dropdown — filtered by selected shape
  • Count — quick buttons (1, 5, 10) or type a number
  • Unit name — label shown above the placed group on the board

Placing a group

  • Click empty space — drops a new group at that position

Selecting & moving

  • Click a base — enters edit mode and selects that base
  • Ctrl + Click — add or remove a base from the current selection
  • Drag a selected base — moves all selected bases together
  • Click empty space — deselects and exits edit mode

Rotating ovals & rectangles

  • Select an oval or rectangle base — a yellow ↻ handle appears above it connected by a dashed line
  • Drag the handle to rotate freely in any direction
  • With multiple bases selected, dragging the handle rotates all selected bases simultaneously
  • Each base stores its own independent rotation — individual bases in a squad can face different directions

Editing a group

  • Right click a base in edit mode — removes that individual base from the group
  • Click a unit's name in the right panel — rename it
  • Click 🗑️ in the right panel — delete the whole group (returns to staging if from an imported list)

Cohesion

  • ■ Green dashed lines = all bases within 2" cohesion
  • ■ Red dashed lines = one or more bases out of cohesion

Other Tools

✏️ Draw Unit

  • Click — add a point to outline a unit zone
  • Right click — undo the last placed point
  • Double click — close the shape and finish
  • Drag inside a finished shape — move it around the board

📏 Measure

  • Click — set the start point, then click again to set the end — distance shown in inches
  • Right click a measurement line — delete it

👁️ Line of Sight

  • Click — set origin point, then click to set target
  • Right click a sight line — delete it

🏷️ Label

  • Click anywhere on the board — places a text label (prompts for text)
  • Right click a label — delete it

Terrain & Missions

Terrain formats

  • GW — Standard GW terrain with 8 layout configurations
  • WTC — Full WTC map images overlaid on the board for all 6 missions
  • UKTC — Coming soon

WTC — Hidden Supplies

When WTC terrain and Hammer & Anvil are both selected, a Hidden Supplies toggle appears in the top nav to switch to the Hidden Supplies variant of that layout.

💾 Save & Load

  • Save Plan — downloads your board state as a JSON file
  • Load Plan — restores a previously saved board from a JSON file
  • Clear Board — resets all drawings, returns list units to staging
  • Your imported army list persists automatically across page refreshes (free users, same browser)
💡 Pro users get cloud save — deployments are stored to your account and accessible from any device or browser.

📱 Mobile

On phones, TacticalDropz runs in view-only mode — the full canvas fills your screen with no toolbar or menus in the way. Use it to reference terrain layouts while setting up the table, or load a saved deployment to show your opponent your plan.

Navigation

  • Tap ☰ in the top-left corner to open the menu
  • Tap outside the menu panel to close it
  • All selections (Format, Mission, Layout) stay open until you close manually

Menu sections

  • Account — log in, sign up, Go Pro, or log out
  • Board — About & Help, Clear Board, Save and load deployments (Pro)
  • Format — switch between GW, WTC, UKTC
  • Mission — select the mission to see deployment zones and objectives
  • Layout — pick terrain layout 1–8
  • Hidden Supplies — appears when WTC + Hammer & Anvil are selected
💡 Tablet users (768px+) get the full desktop experience with all tools.
